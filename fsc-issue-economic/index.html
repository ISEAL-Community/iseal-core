FSC Schema Extension

This page provides a reference specification for the Forest Stewardship Council (FSC) schema extension.

Modules

Metadata elements in FSC schema extension are organized around seven modules:

Assurance Certificate Coverage
Evaluation Form Pesticides
Provenance Scope Status

Metadata Elements

Assurance Module

Elements in the Assurance module: fsc.audit.documentationCatfsc.audit.samplingSystemfsc.audit.sitefsc.audit.stakeholderEngagementfsc.audit.stakeholderTypefsc.audit.type

Certificate Module

Elements in the Certificate module: fsc.certificate.categoryfsc.certificate.type

Coverage Module

Elements in the Coverage module: fsc.coverage.region

Evaluation Module

Elements in the Evaluation module: fsc.evidenceCategory

Form Module

Elements in the Form module: fsc.docCodefsc.standard.type

Pesticides Module

Elements in the Pesticides module: is.pesticide.biocontrolAgentis.pesticide.pestNameis.pesticide.pestSpeciesis.pesticide.pestTypeis.pesticide.activeIngredientis.pesticide.challengeis.pesticide.alternativeMethodis.pesticide.alternativeStrategyis.pesticide.alternativeTrialis.pesticide.alternativeTypeis.pesticide.commonName

Provenance Module

Elements in the Provenance module: fsc.contributor.member

Scope Module

Elements in the Status module: fsc.transitionEndDatefsc.inTransition

Turtle
