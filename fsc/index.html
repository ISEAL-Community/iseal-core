FSC Schema Extension

This page provides a reference specification for the Forest Stewardship Council (FSC) schema extension.

Modules

Metadata elements in FSC schema extension are organized around seven modules:

Assurance Certificate Coverage
Evaluation Form Pesticides
Provenance Scope Status

Status Module

Elements in the Status module: fsc.transitionEndDate fsc.inTransition

 Transition end date
Label Transition end date
Module Status
Definition Transition end date of the standard
Comment Date in YYYY-MM-DD format. At the very least you must enter the year, but month and day is better if possible.
Suggested element fsc.transitionEndDate
Policy Date. Single value only.

 Transition status
Label Transition status
Module Status
Definition Whether the standard is in transition
Comment Yes or No
Suggested element fsc.inTransition
Policy Controlled value. Single select from control list.
Vocabulary vocabulary.txt
